# AI Character Data Master

English: [README.md](README.md)

## 概要

このリポジトリは、画像生成AIやLLMアプリケーションで利用できる、アニメ・ゲーム・漫画などのキャラクターマスターデータを整備するためのものです。

主な目的は以下です。

- キャラクター名の正規化
- Danbooruキャラクタータグの確認
- 原作作品名・出典情報の確認
- キャラクター情報の整理
- 将来的なJSON/CSV出力の元になるマスターデータ作成

## ドキュメント

詳細な仕様は `docs/` 配下を参照してください。

- `docs/research_policy.md`
  - 調査方針、調査フロー、Webアクセス方針、データ構造、自動タスク方針
- `character-master-data/docs/character_gacha_spec.md`
  - キャラ情報フォーマット、ガチャ用データ仕様

## リポジトリ構成

- `data/org_charlist.csv` : 調査用原本（直接編集しない）
- `docs/` : ドキュメント
- `exports/` : 生成・出力データ

## 基本方針

- 原本データは変更しない
- 調査結果は別データとして保存する
- 不確かな情報を推測で確定しない
- 参照元と確信度を記録する
- Web調査は各サイトの方針を尊重し、低頻度かつ必要最小限にする

詳細は `docs/research_policy.md` を参照してください。

## ライセンス・注意事項

キャラクター名、作品名、各種著作権はそれぞれの権利者に帰属します。

このリポジトリは、名称・タグ・作品情報などを整理するための調査用データベースです。
